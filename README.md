## Hijabistahub

Front-End for website [Hijabista Hub](https://hijabistahub.com), for job interview purposes.

## Getting Started

This project require NodeJS v12.

### Install
Install and run this project

```bash
> git clone https://github.com/nizarfadlan/hijabistahub.git
> cd hijabistahub
> npm Install
> npm run dev
```
